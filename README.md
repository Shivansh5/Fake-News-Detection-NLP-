## Run Locally

Clone the project

```bash
  git clone https://github.com/Shivansh5/Real-OR-Fake-News-Detection-NLP-
```

Go to the project directory

```bash
  cd Real-OR-Fake-News-Detection-NLP
```

Install dependencies

```bash
  pip install requirements.txt
```

Start the server

```bash
  flask run app.py
```
